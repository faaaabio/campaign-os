The 1-week sprint

  1. D1
    Brief intake. Story bible + channel plan drafted.
  2. D2
    Shot list. DP writes Seedance prompts; first keyframes via Soul 2.0.
  3. D3
    Generate. Hero beats rendered via Higgsfield MCP; tagged + stored.
  4. D4
    Adapt. Channel variants via HyperFrames. Brand-safety gate.
  5. D5
    Launch. Instrumented from minute 0.
  6. D6
    Measure. Cut + scale variants from in-flight data.
  7. D7
    Retro. Each agent forks prompts/{agent}/v{n+1}.md.

Every render is traceable

File tagging convention is mandatory — performance joins back to the exact prompt version that produced the winning cut.

{campaign-id}_{beat-id}_{channel}_{format}_{variant}_{model}_v{prompt-version}.mp4
